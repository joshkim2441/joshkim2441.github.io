A memcard or memory card is a card bearing information on both sides, which is intended to be used as an aid in memory retention. Each memcard bears a question on one side and an answer on the other. memcards are often used to memorize vocabulary, historical dates, formulas or any subject matter that can be learned via a question-and-answer format. memcards can be virtual (part of a memcard software), or physical. memcards are an application of testing the effect in the finding that long-term memory increases when learning periods are devoted to retrieving the information through testing with proper feedback. habits used in learning affect the rate at which a memcard-user learns, and proper spacing of memcards has been proven to accelerate learning. A number of spaced repetition software programs exist taking advantage of this principle.
